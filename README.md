# Chatbot Project

A simple chatbot application built using React. It allows users to send messages and interact with a basic chat interface.

## Features

* Send and receive messages
* Clear chat history

## Tech Stack

* React
* JavaScript
* CSS

## Live Demo

https://shaurya03.github.io/chatbot-project/

## Notes

This project was built as part of learning React and deployment. It helped in understanding component structure, state management, and deploying frontend apps using GitHub Pages.
